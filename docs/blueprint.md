# **App Name**: Voyage Compass

## Core Features:

- Global Destinations Listing: Browse a curated directory of popular tourist destinations worldwide, with each location having a dedicated page for details.
- Guide Registration & Profiles: Allow local tour guides to create an account, register their services, specify their operational regions (e.g., city, state), and build a public profile.
- Location-Based Guide Discovery: Enable tourists to search for and view registered guides based on specific locations (e.g., 'Amalapuram, Andhra Pradesh').
- AI Local Itinerary Tool: Utilize AI to generate a suggested list of local attractions and a brief itinerary for a selected guide's operational region (e.g., places near Amalapuram).
- Basic Guide Booking Inquiry: Allow tourists to send booking inquiries directly to registered guides through their profiles.

## Style Guidelines:

- Primary color: Muted blue-green (#4E9BAC) to evoke a sense of calm, global horizons, and trustworthy guidance. HSL(190, 40%, 45%).
- Background color: Very light desaturated blue-green (#EEF7F9) providing an open, airy canvas that subtly hints at the primary hue. HSL(190, 15%, 95%).
- Accent color: Vibrant emerald green (#26D9A6) to draw attention to interactive elements and highlight calls to action, providing good contrast. HSL(160, 70%, 50%).
- Headlines font: 'Space Grotesk' (sans-serif) for a modern, slightly technical feel suitable for exploration themes.
- Body font: 'Inter' (sans-serif) for clean, readable, and objective body text on destination descriptions and guide profiles.
- Use clean, minimalist outline icons for navigation, map markers, guide profiles, and booking actions to maintain a professional and intuitive interface.
- Employ a grid-based responsive layout for destination and guide listings, ensuring clear content hierarchy and adaptability across various device sizes. Ample whitespace will enhance readability.
- Incorporate subtle hover effects on interactive cards and buttons, and smooth transitions for page loading to provide a polished user experience without being distracting.
